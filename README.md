Este repositorio busca:

- implementar un ABM reproducible sobre racionalidad limitada y políticas públicas;
- explorar la relación entre satisfacción individual y coordinación colectiva;
- analizar condiciones bajo las cuales emergen:
  - consenso,
  - fragmentación estable,
  - pluralismo persistente,
  - estabilización rápida del sistema;
- ofrecer una base replicable para experimentación computacional y análisis comparativo de parámetros.
